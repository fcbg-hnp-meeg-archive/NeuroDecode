neurodecode.utils.layouts.Layout¶

class neurodecode.utils.layouts.Layout(name, ch_names=None, ch_types=None)¶

Class containing the layout (Cap + AUX) information. Supports: GTEC_16, BIOSEMI_64, SMARTBCI_24, ANTNEURO_64, DSI_24.

Parameters

  • name (str) – The name of the cap. Supported cap names have associated channel names and channel types saved in NeuroDecode.

  • ch_names (list | None) – The list of channels’ name in the order receied from LSL.

  • ch_types (list | None) – The list of channels’ type in the order received from LSL.

  • type. (It provides the layout and channels') –

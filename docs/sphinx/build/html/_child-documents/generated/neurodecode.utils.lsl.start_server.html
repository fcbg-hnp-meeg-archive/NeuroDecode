neurodecode.utils.lsl.start_server¶

neurodecode.utils.lsl.start_server(server_name, n_channels=1, channel_format='string', nominal_srate=0.0, stype='Markers', source_id=None)¶

Start a new LSL server.

Parameters

  • server_name (str) – Name of the server.

  • n_channels (int) – Number of channels.

  • channel_format (str) –

    The channels’ format.
    (‘string’, ‘float32’, ‘double64’, ‘int8’, ‘int16’,

    ’int32’, ‘int64’)

  • nominal_srate (float) – Sampling rate in Hz.

  • stype (str) – Signal type. (https://github.com/sccn/xdf/wiki/Meta-Data#stream-content-types)

  • source_id (str) – Unique identifier of the device or source of the data, if available (such as the serial number). If None, set to server name.

Returns

LSL server object

Return type

LSL outlet

neurodecode.utils.preprocess.dir_current_source_density¶

neurodecode.utils.preprocess.dir_current_source_density(fif_dir, recursive, montage=None, out_dir=None, overwrite=False, **kwargs)¶

Apply a current source density to all raw fif files in a given directory. The file name must respect MNE convention and end with ‘-raw.fif’.

https://mne.tools/stable/generated/mne.preprocessing.compute_current_source_density.html

Parameters

  • fif_dir (str) – The path to the directory containing fif files.

  • recursive (bool) – If true, search recursively.

  • montage (str | DigMontage) – The montage to used, e.g. ‘standard_1020’. c.f. https://mne.tools/stable/generated/mne.io.Raw.html#mne.io.Raw.set_montage

  • out_dir (str | None) – The path to the output directory. If None, the directory f’fif_dir/csd’ is used.

  • overwrite (bool) – If true, overwrite previously corrected files.

  • **kwargs (Additional arguments are passed to) – mne.preprocessing.compute_current_source_density()
